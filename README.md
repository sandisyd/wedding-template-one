# wedding-template-one

This template should help get you started developing with Vue 3 in Vite.


# Fitur yang ada didalamnya
- Cover
- Home Section ( Didalamnya terdapat sambutan, nama pengantin pria dan pengantin wanita serta tanggal pelaksanaan)
- Acara (Berisi foto pengantin pria dan wanita lalu ada informasi orang tua dari masing - masing mempelai dan instagram dari mempelai)
- Informsi Acara (Berisi tentang tanggal pelaksanaan akad dan resepsi beserta lokasi pelaksanaan)
- Love Story (Berisi tentang timeline perjalananan dari pasangan tersebut)
- Save The Date (Berisi tentang counting down dari mulai website sudah dideploy hingga ke tanggal pelaksanaan dan user bisa menyimpan tanggal tersebut ke google calendar)
- Album Foto (Berisi tentang kenangan kenangan saat bersama pasangan)
- Reservation (Berisi tentang reservasi dari para tamu undangan serta ucapan dan doa untuk mempelai)
- Wedding gift (Berisi tentang rekening dari masing - masing mempelai dan alamat dari mempelai jika tamu undangan ingin mengirimkan hadiah)
- Ucapan Penutup (Berisi salam penutup dari kedua mempelai)

# Fitur lainnya
- Akan ada bunga berjatuhan disetiap section
- Akan ada musik sebagai pengantar setelah tamu undangan menekan tombol buka undangan yang ada di cover

# Data reservasi akan disimpan di google sheets. Sehingga ketika client ingin menampilkan ucapan dan doa dari masing - masing tamu undangan itu bisa segera dilakukan

# Route nama tamu undangan tidak bisa diubah sembarangan
